About Bored API

The Bored API suggests random activities to fight boredom. It provides activities with filters for type, participants, price, and accessibility.

This application uses the live API to bring you these activity suggestions in real-time.

How It Works

Select an activity type or leave it as "Any type" for a completely random suggestion.
